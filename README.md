# Unified Handshaking Gateway

The Unified Handshaking Gateway is capable of acting as an intermediary for the following protocols: <br />

- ActivityPub
- Signal
- Discord
- Telegram
- Aardwolf
- Mastodon
- SMS
- Encrypted SMS
- Matrix
- Riot
- Disaspora
- Frendica
- Sneakernet
- RFC 2549
- RFC 1149
- RFC 6241
- BSIP
